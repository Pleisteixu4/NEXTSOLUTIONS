Tag para imagem:

IMG SRC

Para colocarmos uma imagem no nosso html, vocẽ precisa usar a tag "img src", lembre-se de por < e >, um no incio e um no final, após por a tag, coloque o simbolo de igual (=) e após isso, coloque as aspas, dentro das aspas você deve colocar o endereço da imagem, logo a tag ficara como na foto acima.

Tag para ítem:

UL

Esta tag serve para você aplicar ítens ao seu HTML, para aplicar a tag você começara usando as setas das quais foram ditas na descrição anterior, após isso use as siglas UL e após isso, colocara a tag LI, dentro desta tag nós iremos por os itens que queremos, Ex:

    <li> ítem 1 </li>
  • <li> ítem 2 </li>

E por fim, fecharemos a tag com <ul> . Também podemos trocar a tag UL para a OL.

Tag para divisórias:

DIV

Nós usaremos a tag <div> para separarmos conteúdos no nosso HTML, podemos usar elas também para separarmos em classes, um exemplo seria "<div class="">", no CSS conseguiremos mudar as cores e estilo sem mexer no restante do site.

Tag para botão:

BUTTON

Usando a tag <button> nós criaremos um botão e se utilizarmos a tag href, seremos direcionados a outro site clicando neste botão.

Tag de Ancoragem:

HREF

A tag <a href=""> nome do seu arquivo </a> vai te guiar para uma outra pagina, porém precisaremos de um botão para isto. Para alterarmos o estilo do nosso href, devemos por "a" no style (poucos sabem dessa informação).

Tag para formulario:

FORM

Nesta tag, nós criamos um formulario onde ele pode ser preenchido e se utilizarmos um botão, poderemos criar uma pagina de login que realmente funcione. Nesta tag devemos usar o input , usado para criar controles interativos para formulários baseados na web para receber dados do usuário.

Tag para italico/negrito:

STRONG/EM

Na primeira foto vemos uma tag para deixarmos o texto em negrito, é certo para deixar as letras mais fortes na hora da escrita, como se estivesse deixando algo claro, a tag usada seria <strong> e assim teremos um texto em negrito .

Já na segunda foto, temos a tag para deixarmos o texto em italico, signica que deixaremos o texto de forma curvada para o lado, a tag é <em>, assim teremos um texto em italico .

Tag para Cabeçalho:

HEADER

Esta tag serve para fazermos um cabeçalho possivelmente com uma logo, se encontrara sempre no inicio do nosso site, a tag que usaremos será: <header> e como podemos vêr há outra tag na foto acima, esta outra tag serve para separarmos os textos, mais util que a tag <p>, esta tag é a <article>.

Tag para Rodapé:

FOOTER

A tag de <footer> representa um rodapé para o seu conteúdo de seção, algumas informações de copyright ou talvez alguma informação do autor de um artigo. Sempre usaremos esta tag no final do nosso site.

Tag para pseudo-classe:

A utilização de pseudo-classes CSS pode ser uma forma eficaz de tornar a experiência do usuario em uma pagina web mais agradavel e interativa.

HOVER

A tag <hover> serve para quando passarmos o mouse, mude sua forma, apareça uma imagem ou até mesmo troque a cor. Ex:

p: hover {color:green;}

Exemplo de media query no CSS:

p {font-size:40px;}

}

Tag para borda:

BORDER

A tag <border> serve para aplicarmos uma borda em nossas fotos ou até mesmo textos, assim como no exemplo acima. Outro exemplo de tag seria:

p{border: 2px solid blue;}

Tag para marcação:

SECTION

A tag <section> é utilizada para marcar as seções de conteúdo de uma pagina, com este elemento agrupamos de forma logica nosso conteúdo.

O que é a UX?

Primeiramente o que significa UX? UX signica User Experience ou Experiencia do Usuario . User Experience é o design que as equipes usam para criar produtos que fornecem experiências relevantes e significativas para os usuários. Don Norman popularizou o termo “UX”, pois ele acreditava que definições como Interface de Usuário e Usabilidade limitavam o entendimento sobre o que o trabalho dele representava.

Essas estratégias podem incluir pesquisa de usuário, design de Interface, testes de Usabilidade, análise de métricas e feedback contínuo para melhorar a experiênciado usuário. As estratégiasda UX são protegidas para abordar as necessidades dos usuários, compreendendo as suas dores e frustações, e encontrar maneiras de criar soluções que sejam faceis, eficientes e agradaveis de usar.

Condução de entrevista com o usuário

A condução de entrevista é uma técnica que visa entender as necessidades, desejos e experiencias do usuario com produtos ou serviços. Lembrando, durante as entrevistas, é importante estabelecer um ambiente confortavel para os entrevistados e fazer perguntas abertas pra que os incentivem a compartilhar suas experiencias e opniões.

Box-model da UX

Box model é um conjuntor de propriedades que define todos os componentes de uma página web, desde texto até elementos como: botões, imagens, etc. O modelo de caixa é basicamente uma caixa que envolve o conteúdo de um componente, com altura, largura, preenchimento (padding), bordas (border), e margem (margins).

Descrição:

Margem (Margins): Espaços utilizado para movimentar um componente em relação a outro ou limpar uma área fora da borda.

Preenchimento (Padding): Propriedade que pode ser utilizado para aumentar o campo de visual do mesmo, ou afastar um componente filho da borda do componente pai.

Bordas (Border): Bordas são elementos visuais de um componente, seu valores podem ser positivo e nulo.

É importante lembrar que as bordas em excesso podem prejudicar a experiencia do usuario tornando a pagina poluida e dificil de navegar. As bordas tem como objetivo destacar e organizar visualmente o conteudo da sua pagina, tornando-a mais atraente e facil de se ler.

Roteiro de pesquisa

A definição de roteiro de pesquisa na UX é um processo crucial para a obtenção de informações valiosas sobre os usuarios de um produto ou serviço.

Para criar um roteiro de pesquisa de usuarios eficaz, é necessario seguirmos algumas etapas IMPORTANTES . Primeiramente, deve-se definir claramente os objetivos da pesquisa e as informações que deseja obter dos usuarios. Isto pode incluir a identificação de recursos ou funcionalidades que os usuarios desejam em um aplicativo de edição de fotos para smartphones.
